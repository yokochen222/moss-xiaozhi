4HC595引脚连接图：

VCC (16) -------- 3.3V电源
GND (8)  -------- 地线
SER (14) -------- GPIO9 (数据输入)
RCK (12) -------- GPIO10 (锁存时钟)
SCK (11) -------- GPIO11 (移位时钟)
OE (13)  -------- 地线 (输出使能，低电平有效)
MR (10)  -------- 3.3V (主复位，高电平有效)
Q0 (15)  -------- LED0 + 限流电阻(220Ω) + 地线
Q1 (1)   -------- LED1 + 限流电阻(220Ω) + 地线
Q2 (2)   -------- LED2 + 限流电阻(220Ω) + 地线
Q3 (3)   -------- LED3 + 限流电阻(220Ω) + 地线
Q4 (4)   -------- LED4 + 限流电阻(220Ω) + 地线
Q5 (5)   -------- 未使用
Q6 (6)   -------- 未使用
Q7 (7)   -------- 未使用
